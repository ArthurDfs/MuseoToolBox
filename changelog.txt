1.7rc1
----
Lot of changes to satisfy python conventions.

folder and functions names have changed :
    - raster_tools and vector_tools > geo_tools
    - getSamplesFromROI > extract_values
    - historicalMap > load_historical_data
    - getDistanceMatrix > get_distance_matrix

classes now always begin with a capital case : 
    - learnAndPredict > SuperLearn
    - rasterMath > RasterMath
    - sampleExtraction > SampleExtraction
    - sequentialFeatureSelection > SequentialFeatureSelection 

- Fix bug #7 : getSamplesFromROI (renamed extract_ROI) now extracts ROI values using by default memory. If it fails, it will create a temporary raster on disk then delete it when finished. 
- Remove cli
